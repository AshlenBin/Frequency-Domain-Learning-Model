# 频域学习模型（Frequency-Domain Learning Model）

## 介绍
本模型将深度学习算法与傅立叶变换数学原理相结合，旨在解决工程实践中快速傅立叶变换的“频谱泄漏”问题。

## 使用方式
```python
git clone https://github.com/AshlenBin/Frequency-Domain-Learning-Model.git
cd Frequency-Domain-Learning-Model
python FDLM.py
```
